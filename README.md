## Система полиморфных контейнеров 
---

## Описание проекта

Данная лабораторная работа представляет собой реализацию полиморфной системы линейных контейнеров на языке C++ с использованием шаблонов и объектно-ориентированного программирования.

### Основные возможности:

- **Динамический массив** (`DynamicArray<T>`) — массив с изменяемым размером
- **Связный список** (`LinkedList<T>`) — односвязный список
- **Последовательности** (`Sequence<T>`) — унифицированный интерфейс для работы с контейнерами
- **Mutable и Immutable** версии последовательностей
- **BitSequence** — специализированная последовательность для побитовых операций
- **AdaptiveSequence** — адаптивная последовательность, автоматически выбирающая оптимальную реализацию
- **Builder** — паттерн для удобного создания последовательностей
- **Option<T>** — тип для безопасной обработки отсутствующих значений
- **Итераторы** — поддержка интерфейсов `IEnumerator<T>` и `IEnumerable<T>`
- **Map-Reduce** — функциональные операции над последовательностями
- **Перегрузка операторов**
